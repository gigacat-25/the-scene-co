# **App Name**: The Scene Co.

## Core Features:

- Homepage Hero: A visually appealing hero section with the tagline, 'Crafting unforgettable moments, consciously,' and clear CTAs.
- Package Cards: Display event packages (TEDx, Corporate, Other) with descriptions, features, and eco-friendly highlights.
- Contact Form: A comprehensive contact form with name, email, event type, and message fields.
- Sustainability Commitment: Detail the company's eco-friendly practices, materials, and waste reduction approaches.
- Reusable Navbar and Footer: Consistent navigation and footer elements with smooth scrolling and social links.
- Interactive Map: Implement Google Maps for location visibility.

## Style Guidelines:

- Primary color: Deep forest green (#228B22) to evoke nature and tranquility.
- Background color: Off-white (#F8F8FF) provides a clean and minimalist backdrop.
- Accent color: Champagne (#F7E7CE) adds a touch of luxury and elegance.
- Headings: 'Playfair', serif. Body: 'PT Sans', sans-serif.
- Use minimalist icons representing event types and eco-friendly initiatives.
- Incorporate organic shapes and subtle gradients to create a natural, premium feel.
- Subtle fade-in and slide-up effects with Framer Motion to enhance user experience.
